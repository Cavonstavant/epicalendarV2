EpiCalendar V2

Bienvenue sur EpiCalendarV2
Pour synchroniser vos agenda à votre service d'agenda il suffit d'y ajouter les liens suivants:
- https://epicalendarv2.omnirem.dev/{auth}/planning.ics pour le planning Epitech
- https://epicalendarv2.omnirem.dev/{auth}/projects.ics pour le planning de projets Epitech
- https://epicalendarv2.omnirem.dev/{auth}/projects-end.ics pour le planning de rendus de projets Epitech
Remplacez {auth} par votre code d'autologin que vous pouvez trouver à https://intra.epitech.eu/admin/autolog dans le format "auth-jkfsdkjfdsvinn"
